# WHAT DOES THE BOT DO?
Every new member that joins gets a dm, bot asks their stripe email , verifies if they bought subscription, saves in database. Keeps polling, if the subs gets inactive it dms the member, notifies in server and removes the role. After they activate again, the bot adds the role again Can handle multiple subscriptions and products.
